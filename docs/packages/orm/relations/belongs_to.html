Отношение: Belongs To(Пренадлежит к)

Имеет отношение к primary key(первичного ключа) базовой таблицы к полю в своей таблице, относится к одному связанному с ним объекту. Это другая сторона HasOne или HasMany отношений.

Примеры

protected static $_belongs_to = array('post');

Ниже приведены примеры для установления и разрыва belongs to отношений:

// отношения являются новыми:
$comment = new Model_Comment();
$comment->post = new Model_Post();
$comment->save();

// отношения являются уже установленными
$comment = Model_Comment::find(6);
$comment->post = Model_Post::find(1);
$comment->save();

// отношения прекращены
$comment = Model_Comment::find(6);
$comment->post = null;
$comment->save();

Полный пример конфигурации со значениями по умолчанию

// Model_Comment которая belong_to - post
protected static $_belongs_to = array(
	'post' => array(
		'key_from' => 'comment_id',
		'model_to' => 'Model_Post',
		'key_to' => 'id',
		'cascade_save' => true,
		'cascade_delete' => false,
	)
);
